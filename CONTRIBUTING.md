🚀 Guia de Contribuição - Turma de Front-End

Bem-vindo ao nosso projeto prático! Este repositório será nossa base para simular o dia a dia de um desenvolvedor. Siga os passos abaixo para realizar suas atividades.

🛠️ Passo a Passo para Alunos

1. Clonar o Repositório

Abra o terminal na sua pasta de estudos e digite:

git clone [https://github.com/seu-usuario/aula-frontend-grautecnico.git](https://github.com/seu-usuario/aula-frontend-grautecnico.git)
cd aula-frontend-grautecnico


2. Criar sua Branch (Sua área de trabalho)

Nunca faça alterações direto na main. Crie uma ramificação com seu nome: 

git checkout -b aluno-seu-nome


3. Realizar o Desafio

Abra a pasta no VS Code, crie ou edite seus arquivos HTML/CSS conforme as instruções do Professor Alison Melo.

4. Salvar e Enviar suas Alterações

Após terminar o código e salvar os arquivos, rode os comandos:

# Adiciona as mudanças para o envio
git add .

# Cria um rótulo para o que você fez
git commit -m "feat: minha primeira pagina personalizada"

# Envia para o GitHub do Professor
git push origin aluno-seu-nome


⚠️ Regras de Ouro

Semântica: Use as tags corretas (header, section, footer) que aprendemos em aula.
Organização: Mantenha seu código indentado para facilitar a leitura.
Dúvidas: Se travar em algum comando, levante a mão ou verifique a aba de Issues do repositório!
